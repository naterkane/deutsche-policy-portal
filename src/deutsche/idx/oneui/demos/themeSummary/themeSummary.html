Colors used in the "oneui" theme

This section lists color definitions that appear in style rules in the "oneui" theme. If a color is defined with an alpha value (opacity) it is shown after the hex value (and name, if applicable) for the color; if no alpha value is shown, the color definition is opaque.

Colors which are specified for the following style properties are included here:
  • color
  • background (or background-color)
  • border (or border-color, border-top-color, border-bottom-color, border-left-color, border-right-color)

Note that rules which assign values such as "inherited" or "initial" are not included here.

Images used in the "oneui" theme

This section lists images that appear in style rules in the "oneui" theme.

Images which are specified by URL for the following style properties are included here:

  • background (or background-image)
  • border-image (or border-image-source)
  • content
  • list-style (or list-style-image)

Note that rules which assign values such as "inherited", "initial" or "none" are not included here.
